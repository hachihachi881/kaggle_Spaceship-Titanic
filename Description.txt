データセットの説明

このコンペティションでは、宇宙船タイタニック号が時空異常と衝突した際に、乗客が別次元へ転送されたかどうかを予測することが課題です。
予測に役立てるため、損傷した船内コンピュータから回収された乗客の個人記録が提供されています。

ファイルとデータ項目の説明
train.csv

乗客のおよそ3分の2（約8700人分）の個人記録。学習データとして使用します。

PassengerId
各乗客の一意のID。
gggg_pp の形式で、gggg は同じグループの番号、pp はグループ内の個人番号を示します。
同じグループの人は家族であることが多いですが、必ずしもそうとは限りません。

HomePlanet
乗客が出発した惑星（通常は居住惑星）。

CryoSleep
航行中に冷凍睡眠（仮死状態）に入ることを選択したかどうか。
冷凍睡眠中の乗客は自室に留まります。

Cabin
客室番号。
deck/num/side の形式で、side は

P = Port（左舷）

S = Starboard（右舷）

Destination
乗客が下船する予定の惑星。

Age
乗客の年齢。

VIP
航行中に特別VIPサービスを利用したかどうか。

RoomService, FoodCourt, ShoppingMall, Spa, VRDeck
船内の各種高級施設での利用金額。

Name
乗客の氏名（名・姓）。

Transported
別次元へ転送されたかどうか。
これが予測対象（目的変数）です。

test.csv

残り約3分の1（約4300人分）の個人記録。テストデータとして使用します。
このデータに対して Transported を予測するのが課題です。

sample_submission.csv

提出形式のサンプルファイル。

PassengerId
テストデータの乗客ID。

Transported
各乗客について True または False を予測して記入します。
